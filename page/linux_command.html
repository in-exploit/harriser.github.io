Linux目录操作命令

  • cp 拷贝文件

  • cp file1 目录名

    cp file1 file2

  • mv 移动（重命名）文件

  • mv file1 file2

    mv file_catalog file

  • rm 删除文件

  • rm file1（提示是否删除）

    rm -rf file（不提示直接删除）

  • ln 创建文件链接

  • ln s 创建软链接

    ln s filename shortcut_name

  • locate 查找定位文件或目录（模糊查找）

  • locate chmod

  • which 查找出执行命令的绝对路径（精确查找）

  • which cp
