Navicat 12 Activate

所需要下载地址

https://www.navicat.com.cn/products

https://github.com/DoubleLabyrinth/navicat-keygen

1、

根据系统版本，打开对应的navicat-keygen所在文件夹。

2、

执行命令，等待命令结束。

引号内为 Navicat Install Location

navicat-patcher.exe "C:\Program Files\PremiumSoft\Navicat Premium 12"

执行命令，

记录下"Serial number"

navicat-keygen.exe -text ./RegPrivateKey.pem

Select Navicat product，选择Navicat产品，Navicat Premium选"1"

Select product language，选择安装语言

Input major version number，选择Navicat安装版本

Serial number，为随机生成序列号

3、

启动Navicat程序 → 菜单 → 帮助 → 注册。将序列号"Serial number"填入。

关闭网络，然后点击【激活】→ 选择【手动激活】

名字和组织随意填写

① 复制Navicat生成的请求码，粘贴到命令行Input request code in Base64，按2次回车！会生成激活码！

② 将命令行激活码，粘贴回Navicat。

4、将激活码复制到升级框内，点击【激活】，然后激活成功！
